Stream Chat Overlay

A clean, glassmorphism YouTube Live chat overlay for OBS Studio.

▶ Open Overlay for youtube ▶ Open Overlay for twitch GitHub Repo

✨ Features

  • Glassmorphism-style chat bubbles
  • Auto-fading messages
  • Badges: Owner, Mod, Member, Verified
  • Super Chat color tiers
  • Demo mode included
  • Fully OBS-compatible

🚀 How to Use

  • Open OBS Studio
  • Add a Browser Source
  • Set URL to: https://p1xelpp.github.io/stream-chat-overlay/overlay/
  • Set width to 400 and height to 900
  • Enable “Refresh when active”

🛠️ Configuration

Edit overlay/index.html to set:

  • DEMO_MODE = false
  • VIDEO_ID = "your_live_video_id"
  • API_KEY = "your_youtube_api_key"
